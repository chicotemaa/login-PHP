  • ¿Tienes un Usuario? Loguearse

Si dispones de tiempo en estos momentos, nos gustaría hacerte una pequeña prueba en PHP para evaluar tu nivel de conocimiento del lenguaje, así como de bases de datos, patrones de diseño, etc. Para la prueba habíamos pensado la realización de un pequeño proyecto con un formulario de login, que permitiera el login del usuario y una vez logueado se mostrara un pequeño crud donde se listaran los usuarios que hay en el sistema y cualquier usuario logueado pudiera añadir usuarios, y modificar o eliminar los usuarios ya existentes. Para esta prueba usaríamos simplemente PHP, JS y CSS, sin otros frameworks o librerías del lenguaje.
